# hip to be square


## What is it?

A Java library for interacting with the [Atlassian HipChat API v2][api].

## Getting it

Nothing published yet; you'll have to clone and compile yourself:

````
git clone https://github.com/gkopff/h2b2.git
cd h2b2
mvn package
````
